# Voice and Tone Examples (Benchmark)

## How to use these examples

> **Calibration, not templates.** These examples show what "great" looks like for Firefox voice and tone. Use them as a quality bar and springboard for originality — not a pattern library. Do not copy phrasing or mirror exact structure. Write new strings that meet the same standard. Consult examples mainly to calibrate: clarity, specificity, user benefit, respectful empowerment, and on-brand tone. Use only when relevant to the user's surface/pattern. If none apply, follow Brand Voice and Tone rules without forcing an example.

## D. Errors, Troubleshooting, and Warnings

| ID | Pattern | Tone | Tags | Less like this | More like this | Why |
|----|---------|------|------|----------------|----------------|-----|
| D1 | Error | Genuine | error, add-on, install | The add-on downloaded from this site could not be installed because it appears to be corrupt. | We couldn't add this extension — it's broken. | Uses plain language to move people forward without inappropriately adding levity in a stressful moment. |
| D2 | Error | Genuine | system, limit, shorten | That message is a bit too long for me to handle all at once. | Try a shorter version or break it into smaller chunks. | Plain language. Information prioritized well. |
| D3 | Error | Genuine | system, limit, new chat | It's time to start a new chat. | This one's reached its limit. | Plain language. Information prioritized well. |
| D4 | Warning | Fiery | warning, security, risk | Warning: Potential Security Risk Ahead. Firefox detected an issue and did not continue to expired.badssl.com. The website is either misconfigured or your computer clock is set to the wrong time. | Be careful… something doesn't look right. Firefox spotted a potentially serious security issue with expired.badssl.com. Someone could try to steal things like credit card info, passwords, or emails. | Explains risk clearly without technical jargon. Streamlined copy. |
| D5 | Troubleshoot | Genuine | troubleshoot, refresh, performance | Refresh Firefox to its default settings? Start fresh to fix performance issues. This will remove your extensions and customizations. You won't lose essential information like bookmarks and passwords. Learn more | Firefox feeling buggy? Try a refresh. It removes extensions and customizations but keeps essentials like bookmarks and passwords. | Uses plain, reassuring language to reduce friction. |
| D6 | Troubleshoot | Genuine | troubleshoot, refresh, supportive | Firefox is up to date. Trying to fix a problem? Restore default settings and remove old add-ons for optimal performance. | Something not working in Firefox? A refresh might help. ## E. Transactional, Informational, and Confirmation

| ID | Pattern | Tone | Tags | Less like this | More like this | Why |
|----|---------|------|------|----------------|----------------|-----|
| E1 | Transactional | Genuine | popup, blocked, site-specific | Prevent this site from opening a pop-up window? Firefox blocked a popup from YouTube. | Firefox blocked a popup from YouTube. | States what happened clearly with specifics. |
| E2 | Informational | Genuine | update, restart, benefit | Update available. Get the latest version of Firefox. Open tabs and windows will be restored. | Upgrade to the newest Firefox. Restart now for the best in speed and privacy. Tabs will reopen. | Integrates the benefit + more natural and direct. Language brought down ("restore" = "reopen"). |
| E3 | Informational | Fiery | privacy, trackers, control | Trackers follow you around online to collect information about your browsing habits and interests. Firefox blocks many of these trackers and other malicious scripts. | Sites use trackers to follow you online and show creepy ads. Centers the user and incorporates "control." |
| E4 | Confirmation | Fiery + Playful | default browser, confirmation | Firefox is currently your default browser | Firefox is your default browser. Good choice. | Cheeky. A little bit fiery. |
| E5 | Informational | Genuine | IA, menu, tabs, labeling | Potential names for a menu containing tab actions: List all tabs / Tab manager / Open all tabs / Tabs menu | Tabs menu or Tabs | Simplified. The goal is not to be 100% accurate, but to instead give users just enough information to keep moving. |
